# 🔐 Login System with Tkinter

This project is a simple graphical user interface (GUI) login and registration system built using Python and Tkinter.

It allows users to:

Register a new account
Log in with existing credentials
Receive feedback for success or failure actions

## 🚀 Features
 - Login validation
 - User registration
 - Password validation (minimum length)
 - Error handling with popup messages
 - Clean GUI using Tkinter and ttk

## 🖥️ Technologies Used
 - Python 3
 - Tkinter (GUI library)

## 📂 Project Structure
 - main.py
 - .gitignore
 - README.md
 
## ⚠️ Limitations
 - User data is stored in memory (not persistent)
 - No encryption for passwords
 - Not suitable for production use
## 💡 Future Improvements
 - Store users in a database (SQLite)
 - Hash passwords for security
 - Improve UI design
 - Add session management
